# 🐾 Pet Adoption System

A web-based Pet Adoption System developed to help users browse, apply, and adopt pets online.

## 📌 Project Description
This project is designed to make pet adoption easier and more organized.

### Users can:
- View available pets
- Apply for pet adoption
- Check application status

### Admin can:
- Add new pets
- Manage pet details
- Approve or reject adoption applications

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## ⚙️ Main Features
- View pets
- Adoption application form
- Application status checking
- Admin dashboard
- Pet management
- Application approval/rejection

## 🚀 How to Run
1. Download or clone the project
2. Place it inside the XAMPP `htdocs` folder
3. Start Apache and MySQL in XAMPP
4. Import the database into phpMyAdmin
5. Open the project in your browser
